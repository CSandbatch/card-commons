# Help test a smaller creative primitive.

Card Commons is a proposal with working contracts and a working Card Studio—not
a finished platform. The Studio creates the first card type today, with AI image
generation verified live across six models; it runs as a private pilot while the
rest of the loop is built. The next useful step is evidence from people who
build, create, collect, teach, publish, and play.

## A useful first contribution

1. Read the [executive brief](../specs/00-executive-brief/).
2. Inspect the [protocol and fixtures](https://github.com/CSandbatch/card-commons/tree/main/contracts).
3. Open a focused issue with a task, incompatibility, source, or prototype.
4. Keep verified evidence separate from an attractive hypothesis.
